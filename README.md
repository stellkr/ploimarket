## 1. 프로젝트 개요
**PloiMarket**은 개발자와 디자이너를 위한 고품질 웹 템플릿 및 IT 기기를 판매하는 가상의 종합 디지털 마켓플레이스입니다. 트렌디한 "Modern Tech" 스타일을 표방하며, 사용자 친화적이고 직관적인 웹 인터페이스를 구현하는 것에 중점을 두었습니다. 

## 2. 주요 디자인 및 레이아웃 컨셉
- **색상 팔레트**: 신뢰감을 주는 네이비(`#2c3e50`)를 메인 컬러로 사용하고, 배경은 밝은 라이트 그레이와 화이트를 조합하여 콘텐츠(상품)가 돋보이는 미니멀한 디자인을 채택했습니다.
- **레이아웃**: `CSS Grid`와 `Flexbox`를 적극 활용하여, 다양한 화면 크기에 유연하게 대응할 수 있는 반응형(Responsive) 카드 레이아웃을 구축했습니다.
- **사용자 경험(UX)**: 부드러운 전환 효과(Transition) 및 페이드인(Fade-in) 애니메이션을 적용하여 딱딱하지 않고 동적인 사용자 경험을 제공합니다. 메인 페이지에는 핵심 정보를 롤링하는 **이미지 슬라이더(Carousel)**를 구현하였습니다.

## 3. 주요 페이지 및 기능 설명
본 프로젝트는 다음과 같은 총 8개의 유기적으로 연결된 페이지로 구성되어 있습니다.

1. **index.html (메인 페이지)**
   - 최신 프로모션 배너 슬라이더 적용 (JavaScript 기반 자동 슬라이드 및 좌우 컨트롤 제어)
   - 주요 상품 카테고리(IT 기기, 웹 템플릿) 필터링 탭 UI 적용
   - 전체적인 내비게이션 허브 역할 수행

2. **best.html (베스트 상품 페이지)**
   - 평점과 판매량이 높은 인기 상품들을 랭킹 배지(1위, 2위 등)와 함께 강조하여 노출

3. **it-devices.html / web-templates.html (카테고리 상세 페이지)**
   - 특정 상품군(하드웨어 및 소프트웨어 에셋)에 집중된 상품 나열 및 모달(Modal) 창을 통한 상세 정보 '미리보기' 제공

4. **about.html (회사 소개) / cs.html (고객센터)**
   - 서비스의 철학을 설명하는 랜딩 페이지와 사용자의 문의를 접수할 수 있는 폼(Form) UI 구현

5. **login.html / signup.html (인증 페이지)**
   - 직관적인 카드형 폼 디자인 적용
   - JavaScript를 활용한 클라이언트 사이드 **유효성 검사(Validation)** 구현 (아이디/비밀번호 누락, 체크박스 확인 등)

## 4. 핵심 구현 기술
- **HTML5**: 시맨틱 태그(`<header>`, `<main>`, `<section>`, `<footer>`)를 준수하여 웹 접근성 및 구조 향상
- **CSS3**: 
  - 공통 변수(`:root`)를 활용한 일관된 테마(색상, 둥근 모서리, 그림자 효과) 관리
  - 미디어 쿼리와 Grid/Flex를 결합한 반응형 웹 구조
- **JavaScript**: 
  - 모달(Modal) 팝업창 제어
  - 메인 페이지 배너 슬라이더 인터랙션
  - 회원가입/로그인 양식 검증(Form Validation) 및 동적 에러 메시지 노출
  - 현재 페이지에 따른 네비게이션 메뉴(`GNB`) 활성화 상태(`active` 클래스) 자동 부여

## 5. 마치며
기획부터 레이아웃 설계, 그리고 디테일한 CSS 스타일링과 자바스크립트 동적 제어까지 전 과정을 직접 다루어보며 웹 프론트엔드 개발에 대한 전반적인 이해도를 크게 높일 수 있었습니다. 특히 페이지 간 일관성을 유지하기 위해 공통 CSS 클래스를 설계하고 버그를 잡아가는 과정에서 웹 디자인 시스템의 중요성을 배웠습니다.﻿# ploimarket
